## 📖 Deskripsi Aplikasi

**Internship Task Tracker** adalah sebuah aplikasi web sederhana berbasis HTML, CSS, dan JavaScript yang dikembangkan untuk melacak dan memantau progres tugas magang bersama tim. Aplikasi ini mempermudah kolaborasi dalam mendata pekerjaan dan anggota yang bertanggung jawab atas setiap tugas secara interaktif.

## ✨ Fitur Aplikasi

- **Tambah Tugas Baru:** Pengguna dapat menambahkan tugas-tugas yang perlu diselesaikan.
- **Penugasan (Assignee):** Terdapat kolom opsional untuk mencatat siapa anggota tim yang bertanggung jawab mengerjakan tugas tersebut.
- **Daftar Tugas:** Menampilkan daftar tugas yang telah ditambahkan.
- **Penghitung Tugas:** Menampilkan jumlah total tugas yang ada dalam daftar.
- **Antarmuka Interaktif:** Desain antarmuka (UI) yang bersih dan mudah digunakan berkat CSS.

## 🚀 Cara Menjalankan Program

Karena ini merupakan aplikasi *front-end* statis sederhana, Anda bisa langsung membukanya di browser:

1. Clone atau unduh (*download*) repository/project program ini ke komputer Anda.
2. Buka folder dari project tersebut (misalnya `todolist-html-main`).
3. Cari file `index.html`.
4. Klik ganda (double-click) pada file `index.html` tersebut untuk membukanya secara otomatis di web browser bawaan Anda (Chrome, Firefox, Safari, Edge, dll).
5. (Opsional) Jika menggunakan *Visual Studio Code*, Anda bisa menggunakan ekstensi **Live Server** dan klik `Go Live` untuk membukanya local server.
